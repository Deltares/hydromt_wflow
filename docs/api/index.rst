#############
API reference
#############

The HydroMT-Wflow API provides classes and methods to build, update, and analyze
Wflow models. The API is divided into the following sections:

- :ref:`SBM model <api_wflowsbmmodel>`: Class for building and managing SBM Wflow models.
- :ref:`Sediment model <api_wflowsedimentmodel>`: Class for building and managing sediment transport models.
- :ref:`Advanced API <api_reference_advanced>`: In-depth information, scientific motivations and advanced usage.
